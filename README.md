## 介绍
QuarkJar是一个基于SpringBoot的低代码工具；它提供的丰富组件，能帮助您使用很少的代码就能搭建出功能完善的应用系统。

## 题外话
我知道在Java界，有着“若依”这种类似的框架，而且使用的用户也比较多；写QuarkJar的初衷并不是想取代“若依”，而是希望为开发者提供更广阔的思路、更多的选择。

## 系统特性

- 用户管理
- 权限系统
- 菜单管理
- 系统配置
- 操作日志
- 附件管理
- 组件丰富

## 快速开始
