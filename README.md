## 介绍
QuarkJar是一个基于SpringBoot的低代码工具；它提供的丰富组件，能帮助您使用很少的代码就能搭建出功能完善的应用系统。

## 系统特性

- 用户管理
- 权限系统
- 菜单管理
- 系统配置
- 操作日志
- 附件管理
- 组件丰富

## 快速开始
启动服务，访问后台地址： ```http://127.0.0.1:8080/admin/```

账号：```administrator```
密码：```123456```
